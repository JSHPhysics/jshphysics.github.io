01 / Astrophysics ↗

Stellar Parallax Lab

Follow a guided walkthrough of Stellar Parallax using real images. Select from 5 stars and use their real images and data to determine how far away they are.

A-Level
02 / Nuclear ↗

Radioactive Decay Lab

Explore radioactive decay with adjustable half-lives — including a dice view and a nuclei view with live graph plotting and exportable data so students can analyse it themselves.

GCSE A-Level
03 / Mechanics ↗

Upthrust Derivation

A walkthrough derivation of Archimedes' upthrust from first principles. Each stage builds up interactively with students answering self-marking questions as they progress.

Revision Tracker (preview)

A multi-subject GCSE & A-level revision checklist. Track progress across topics and see what's left to do. The full app is in development — this is an early preview. If you would like your subject added simply give me a checklist of the spec points and I'll see what I can do.

Clock Cipher

A live encode/decode tool with auto-snap rotation. Good for cryptography starters, end-of-term puzzles, or just having a fiddle.

A note
on these
tools

These tools and simulations are built with the help of Claude Code. I have some coding experience, but these wouldn't be possible — at least not at this pace or polish — without it.

That means there may occasionally be errors, either in the underlying physics or in the code itself. I've done as much testing as I can to make sure that isn't the case, but I'd genuinely welcome anyone who finds something off — a bug, a misleading explanation, a value that looks wrong, or just an idea for how a tool could be improved — to submit an issue on the relevant GitHub repository.

You can find every repo listed at github.com/JSHPhysics. Open the repo for the tool in question and use the Issues tab. Improvements, corrections, and even small typo fixes are all appreciated.
